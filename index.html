Cheapest AI Models

Real-time per-token pricing for LLMs & coding models, sorted cheapest → most expensive. Data is fetched live on page load from the OpenRouter models API.

Quality Design Arena ELO — a coding/UI skill rating from head-to-head match-ups (like chess). Higher = better (here ≈ 900–1400). “—” = not benchmarked (~⅓ of models).
Balance 0–100 value score — 100 = best, 0 = worst (relative to models shown). Blends price & quality; set the mix with the Cheapest ↔ Best slider.

# Model Company Input $/1M Output $/1M Blended $/1M Quality Balance Context

How to read this: prices are USD per 1M tokens (per-token rates × 1,000,000), pulled live from OpenRouter. Blended is an estimated effective rate assuming a mix of input/output tokens (default weighting 3 parts input : 1 part output) — change it with the “Sort cost basis” selector. Click any column header to re-sort.

Quality is the average Design Arena ELO (a coding/design-output benchmark) that OpenRouter exposes — only about a third of models have it; the rest show “—”. Balance blends price and quality into a 0–100 score: drag the Cheapest ↔ Best slider to weight it (left = cheapest, right = highest quality), which ranks the table by Balance. Models without a quality score are treated as unrated.

⚠️ This compares per-token model pricing. Subscription-based coding agents (e.g. Warp, GitHub Copilot, Cursor, Codex CLI) are billed by plan rather than per token, so they aren't directly comparable — see them on the separate Subscription agents page. Pricing changes frequently — always confirm with the provider before relying on it.
